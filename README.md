# 3.5.2.Functional_programming_in_Scala
# Binārais meklēšanas koks 

Programma nolasa skaitļu skaitu `n` un pēc tam ievada `n` veselu skaitļu virkni.<br>
<br>
No ievadītajiem skaitļiem tiek izveidots binārais meklēšanas koks, secīgi pievienojot elementus tieši tajā pašā kārtībā, kādā tie tika ievadīti.<br>
<br>
Pēc koka izveides programma nolasa veselu skaitli `x` un meklē to kokā.<br>
<br>
Ja skaitlis `x` kokā netiek atrasts, programma izvada paziņojumu `not found`.<br>
<br>
Ja skaitlis `x` kokā ir, programma izvada maršrutu no saknes līdz mezglam ar `x`, izmantojot simbolu kombināciju no `*`, `L` un `R` ( `*` - sakne, `L` - pāreja uz kreiso zaru, `R` - pāreja uz labo zaru).
